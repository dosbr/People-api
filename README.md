# people-api

## Tecnologias
- Java
- Maven
- Springboot
- SpringDoc
- Junit
- Mockito
- Lombok

## Documentação
- `http://localhost:8080/swagger-ui/index.html`

## Executar
> Precisa do maven e java jdk 17 instalado 
1. `git clone https://github.com/dosbr/people-api.git`
2. `cd people-api`
3. `mvn clean package`
4. `cd target`
5. `java -jar api-0.0.1-SNAPSHOT.jar`
